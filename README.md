# CRC (Cyclic Redundancy Check) Implementation in C++

CRC (Cyclic Redundancy Check) algorithms in C++, covered by tests.

## Test

To run tests,

```bash
mkdir build && cd build && cmake .. && cmake --build . && ctest
```

## Lib

1. ``Lib_crc8``

2. ``Lib_crc16``

3. ``Lib_crc32``

4. ``Lib_crc_in_vtm``
